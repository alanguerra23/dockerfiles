# Dockerfiles

## Instalação/Uso

```bash
git clone https://github.com/als-guerra/dockerfiles.git $HOME/.dockerfiles
```

```bash
cp $HOME/.dockerfiles/templates/<nomedotemplate>/{docker-compose.yml,dockerfile,,dockerignore} caminho-seu-projeto/
```
